Global Terrorism Database

The following dataset imports the data Alex Newhouse used for his SQL Geekout Presentation. Click the download button and then paste the command below it in your Terminal (Mac Users) or Command Prompt (Window Users).

  • --db gtdb tells the MongoServer to import the data into a db called gtdb
  • --collection terror tells the MongoServer that within the gtdb database import the data into a collection called terror
  • --type csv tells the MongoServer that it is a csv file extension
  • --headerline tells the MongoServer the data source has a Header row
  • --file tells the MongoServer where on your computer is the file located. In my case my csv file is located in my Downloads directory (i.e., "C:/Users/carlo/Downloads/gtdb.csv")

Download Data

mongoimport --db gtdb --collection terror --type csv --headerline --file C:/Users/carlo/Downloads/gtdb.csv
